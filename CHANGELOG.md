# Changelog

本项目的所有重要变更都会记录在此文件中。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，并建议遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [0.0.3-beta] - 2026-07-23

### Added
- 新增多人共享事件“奇怪的空间”，支持全队统一选项。
- 新增事件选项“伸手去取”：获得药水“刻板印象”。
- 新增事件选项“观看影像”：从3张随机多人卡中选择1张获得。
- 新增药水“刻板印象”，可按角色生成本回合免费形态牌。
- 新增5张形态牌与对应5个形态能力：原神形态、运转形态、慷慨形态、发紫形态、相信形态。
- 新增猎人稀有能力“蛇神形态”：攻击伤害同时给予等量中毒。
- 新增储君稀有能力“星神形态”：攻击造成伤害时获得1辉星。
- 新增战士稀有能力“瓦库形态”：回合开始获得1能量、回复最大生命百分比，并由瓦库接管本回合；升级提升回复比例。
- 新增机器人稀有能力“机神形态”：激发充能球时获得1点临时集中。

### Changed
- 更新版本号至 `0.0.3-beta`。
- 慷慨形态 / 相信形态改为发放原版升级牌，并附加消耗。
- 发紫形态改为正确使用原版灾厄（DoomPower）。

### Fixed
- 修复原神形态仅变化手牌、未覆盖弃牌堆与消耗堆的问题。
- 修复慷慨形态由队友攻击触发的铸造不会给队友塞牌的问题。
- 修复瓦库形态能量加成不生效（改为与薪火之源相同的最大能量修正）。
- 修复“观看影像”在多人模式下所有人看到相同卡牌选项的问题。
- 修复英文本地化 `powers.json` 缺少逗号导致加载失败的问题。

## [0.0.2-beta] - 2026-07-19

### Added
- 新增事件“被诅咒的细胞”与对应双选项结算流程。
- 新增事件遗物“被诅咒的细胞”“细胞再生”与诅咒卡“诅咒之刃+”。
- 新增事件“奇怪的神秘人”。（背景图来自王筱巫）
- 新增更多玩偶联动事件“稻草人与玫瑰”。
- 新增“全员坠机”卡图（来自铃铛蔷薇）

### Changed
- “全面攻杀”移除消耗。
- “全员坠机”移除消耗。升级效果每多一位玩家增加24点伤害上调至26点。
- 事件图片加载逻辑简化为统一基类自动按事件类名寻找资源并回退默认图。
- 多语言文案更新：同步事件、遗物与相关描述到当前机制表现。

### Fixed
- 修复“别样的商人”贷款选项缺失贷款卡悬浮提示的问题。

### Added

### Fixed
