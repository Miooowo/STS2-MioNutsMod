# Changelog

本项目的所有重要变更都会记录在此文件中。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，并建议遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [0.0.2-beta] - 2026-07-19

### Added
- 新增事件“被诅咒的细胞”与对应双选项结算流程。
- 新增事件遗物“被诅咒的细胞”“细胞再生”与诅咒卡“诅咒之刃+”。
- 新增事件“奇怪的神秘人”。（背景图来自王筱巫）
- 新增更多玩偶联动事件“稻草人与玫瑰”。
- 新增“全员坠机”卡图（来自铃铛蔷薇）

### Changed
- “全面攻杀”移除消耗。
- “全员坠机”移除消耗。升级效果每多一位玩家增加24点伤害上调至26点。
- 事件图片加载逻辑简化为统一基类自动按事件类名寻找资源并回退默认图。
- 多语言文案更新：同步事件、遗物与相关描述到当前机制表现。

### Fixed
- 修复“别样的商人”贷款选项缺失贷款卡悬浮提示的问题。

## [0.0.1.1-beta] - 2026-07-18

### Added
- 新增“别样的商人”事件专用背景图资源 `odd_merchant_event.png`。

### Changed
- 事件图片加载路径统一接入 `STS2-MioNutsMod/images/events` 目录，新增统一事件资源路径工具。
- “别样的商人”事件改为通过统一路径加载事件背景，便于后续事件复用。

### Fixed
- 修复事件图片路径分散导致的资源维护不一致问题，统一缺失资源时回退到 `mod_image.png`。

## [0.0.1-beta] - 2026-07-18

### Added
- 新增事件“钢铁和巨兽”，并补充多人场景下的选项流程与奖励结算。
- 新增多张多人卡牌：与我饮血、全力攻杀、运球时间、你不许转、全员坠机、盗墓团伙、传道受业。
- 新增衍生卡与能力：死亡收割、死吧死吧死吧、同调充能、传道受业等配套实现。
- 新增相关卡图/能力图资源与本地化条目（中文/英文）。

### Changed
- 调整多张卡牌数值、升级效果与多人约束，补全卡面动态数值显示。

### Fixed
- 修复多人事件进入战斗的同步异常与非共享事件冲突问题。
- 修复部分卡牌创建时无 owner 导致的运行时错误。
