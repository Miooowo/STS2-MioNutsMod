# STS2-MioNutsMod 0.0.3-beta

发布日期：2026-07-23

## Added
- 新增多人共享事件“奇怪的空间”，支持全队统一选项。
- 新增事件选项“伸手去取”：获得药水“刻板印象”。
- 新增事件选项“观看影像”：从3张随机多人卡中选择1张获得。
- 新增药水“刻板印象”，可按角色生成本回合免费形态牌。
- 新增5张形态牌与对应5个形态能力：原神形态、运转形态、慷慨形态、发紫形态、相信形态。
- 新增猎人稀有能力“蛇神形态”：攻击伤害同时给予等量中毒。
- 新增储君稀有能力“星神形态”：攻击造成伤害时获得1辉星。
- 新增战士稀有能力“瓦库形态”：回合开始获得1能量、回复最大生命百分比，并由瓦库接管本回合；升级提升回复比例。
- 新增机器人稀有能力“机神形态”：激发充能球时获得1点临时集中。

## Changed
- 更新版本号至 `0.0.3-beta`。
- 慷慨形态 / 相信形态改为发放原版升级牌，并附加消耗。
- 发紫形态改为正确使用原版灾厄（DoomPower）。

## Fixed
- 修复原神形态仅变化手牌、未覆盖弃牌堆与消耗堆的问题。
- 修复慷慨形态由队友攻击触发的铸造不会给队友塞牌的问题。
- 修复瓦库形态能量加成不生效（改为与薪火之源相同的最大能量修正）。
- 修复“观看影像”在多人模式下所有人看到相同卡牌选项的问题。
- 修复英文本地化 `powers.json` 缺少逗号导致加载失败的问题。
